
 Web Project
 
 Project 2 - HTML Foundation Page

For this project, I created a basic HTML website focused on laser engraving. The purpose of the site is to explain what laser engraving is, show different materials that can be used, and give examples of projects that can be created. I chose this topic because it is something creative and practical that can be used for both personal projects and small business ideas.

The website includes two main pages. The home page (index.html) introduces the topic and provides navigation to the second page. The second page (project2.html) goes more in depth and includes paragraphs, lists, a table, images, and links related to laser engraving.

While working on this project, I learned how to build a webpage using only HTML. I practiced using important elements like headings, paragraphs, lists, images, links, and tables. I also learned how to connect pages together using navigation links and how to organize files into the correct folders.

One challenge I had was making sure all of my links worked correctly between pages. I also had to fix errors from the W3C validator, such as adding the correct language tag, character encoding, and removing outdated HTML attributes. Once I fixed those, my pages validated correctly.

Overall, this project helped me understand the structure of a webpage and how everything connects together before adding styling and more advanced features later on.

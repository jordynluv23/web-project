# web-project
## Project 2 - HTML Foundation Page

For Project 2, I created a webpage about laser engraving. My page explains what laser engraving is, what materials can be used, and some examples of projects that can be made with a laser engraver.

While creating this page, I learned how to build a basic HTML webpage using headings, paragraphs, lists, images, links, tables, and navigation links. I also learned how important it is to save files in the correct folders so the links work properly.

One challenge I had was making sure the navigation worked from the index page to the project2 page. I also had to pay attention to the image link and alt text so the page would be more accessible.
